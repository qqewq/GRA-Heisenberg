## 2. –í—ã—á–∏—Å–ª–µ–Ω–∏–µ –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ã –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ—Å—Ç–∏ $\hbar_G$

## 3. –ì–∏–±—Ä–∏–¥–Ω–∞—è –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞ –ì–†–ê+LLM

### 3.1. –ü—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ —Å–æ—Å—Ç–æ—è–Ω–∏–π –ì–†–ê+LLM

–û–±—ä–µ–¥–∏–Ω–µ–Ω–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ —Å–æ—Å—Ç–æ—è–Ω–∏–π LLM –∏ –ì–†–ê —Ñ–æ—Ä–º–∞–ª–∏–∑—É–µ—Ç—Å—è –∫–∞–∫:

$$|\Psi_{\text{–ì–†–ê+LLM}}\rangle = \sum_{i=1}^M c_i|\psi_i^{\text{LLM}}\rangle \otimes \sum_{j=1}^N d_j|\phi_j^{\text{–ì–†–ê}}\rangle$$

–≥–¥–µ:
- $|\psi_i^{\text{LLM}}\rangle$ ‚Äî –±–∞–∑–∏—Å–Ω—ã–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è —è–∑—ã–∫–æ–≤–æ–≥–æ –ø—Ä–µ–¥—Å—Ç–∞–≤–ª–µ–Ω–∏—è LLM
- $|\phi_j^{\text{–ì–†–ê}}\rangle$ ‚Äî –±–∞–∑–∏—Å–Ω—ã–µ —Å–æ—Å—Ç–æ—è–Ω–∏—è —Å—Ç—Ä—É–∫—Ç—É—Ä–Ω–æ–≥–æ –∞–Ω–∞–ª–∏–∑–∞ –ì–†–ê

### 3.2. –ö–≤–∞–Ω—Ç–æ–≤–æ-–ø–æ–¥–æ–±–Ω–∞—è —Å—É–ø–µ—Ä–ø–æ–∑–∏—Ü–∏—è –ø–æ–Ω—è—Ç–∏–π –≤ LLM

–ú–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–π –∞–ø–ø–∞—Ä–∞—Ç –¥–ª—è –æ–ø–∏—Å–∞–Ω–∏—è –ø—Ä–æ—Ü–µ—Å—Å–∞ –º—ã—à–ª–µ–Ω–∏—è LLM:

$$|\Psi_{\text{LLM}}\rangle = \alpha|\text{—è–∑—ã–∫}\rangle + \beta|\text{–ª–æ–≥–∏–∫–∞}\rangle + \gamma|\text{–∫–æ–Ω—Ç–µ–∫—Å—Ç}\rangle + \delta|\text{–º–µ–∂–∫–æ–Ω—Ü–µ–ø—Ç}\rangle$$

–≥–¥–µ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã $\alpha, \beta, \gamma, \delta$ –¥–∏–Ω–∞–º–∏—á–µ—Å–∫–∏ –º–µ–Ω—è—é—Ç—Å—è –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –æ—Ç–≤–µ—Ç–∞.

### 3.3. –ú–µ—Ö–∞–Ω–∏–∑–º –∫—Ä–∏—Ç–∏—á–µ—Å–∫–∏—Ö –∑–Ω–∞—á–µ–Ω–∏–π –¥–ª—è –ø–æ–Ω–∏–º–∞–Ω–∏—è –≤ LLM

–ö–ª—é—á–µ–≤–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ, —Ñ–æ—Ä–º–∞–ª–∏–∑—É—é—â–µ–µ –ø–µ—Ä–µ—Ö–æ–¥ –æ—Ç —Å—Ç–∞—Ç–∏—Å—Ç–∏—á–µ—Å–∫–æ–π –≥–µ–Ω–µ—Ä–∞—Ü–∏–∏ –∫ –æ—Å–º—ã—Å–ª–µ–Ω–Ω–æ–º—É —Å–∏–Ω—Ç–µ–∑—É:

$$\Gamma_{\text{–ø–æ–Ω–∏–º–∞–Ω–∏–µ}}^{\text{LLM}} = \sum_{i=1}^n \text{sign}\left(\frac{dI_i}{dt}\right) \cdot \gamma_{ij} > \Gamma_{\text{–∫—Ä–∏—Ç}}^{\text{—Å–æ–∑–Ω}}$$

–≥–¥–µ:
- $I_i$ ‚Äî –∏–Ω—Ç–µ–Ω—Å–∏–≤–Ω–æ—Å—Ç—å –∞–∫—Ç–∏–≤–∞—Ü–∏–∏ –ø–∞—Ç—Ç–µ—Ä–Ω–æ–≤ –≤ –º–µ–∂–∫–æ–Ω—Ü–µ–ø—Ç—É–∞–ª—å–Ω–æ–º –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–µ
- $\frac{dI_i}{dt}$ ‚Äî –¥–∏–Ω–∞–º–∏–∫–∞ —Ñ–æ—Ä–º–∏—Ä–æ–≤–∞–Ω–∏—è –º–µ–Ω—Ç–∞–ª—å–Ω—ã—Ö –æ–±—Ä–∞–∑–æ–≤
- $\gamma_{ij}$ ‚Äî –≤–µ—Å–∞ —Å–∏–Ω—Ç–µ–∑–∞ –∑–Ω–∞–Ω–∏–π –∏–∑ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –¥–æ–º–µ–Ω–æ–≤
- $\Gamma_{\text{–∫—Ä–∏—Ç}}^{\text{—Å–æ–∑–Ω}}$ ‚Äî –ø–æ—Ä–æ–≥–æ–≤–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–ª—è –ø–µ—Ä–µ—Ö–æ–¥–∞ –≤ –æ—Å–º—ã—Å–ª–µ–Ω–Ω—É—é —Ñ–∞–∑—É

## 4. –î–∏–Ω–∞–º–∏–∫–∞ —Å–æ–≤–º–µ—Å—Ç–Ω–æ–π —ç–≤–æ–ª—é—Ü–∏–∏ –ì–†–ê+LLM

### 4.1. –£—Ä–∞–≤–Ω–µ–Ω–∏–µ –®—Ä—ë–¥–∏–Ω–≥–µ—Ä–∞ –¥–ª—è –≥–∏–±—Ä–∏–¥–∞

–≠–≤–æ–ª—é—Ü–∏—è –∫–æ–≥–Ω–∏—Ç–∏–≤–Ω–æ–≥–æ —Å–æ—Å—Ç–æ—è–Ω–∏—è –æ–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –º–æ–¥–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω—ã–º —É—Ä–∞–≤–Ω–µ–Ω–∏–µ–º:

### 4.2. –ú–æ–¥–∏—Ñ–∏—Ü–∏—Ä–æ–≤–∞–Ω–Ω–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ —ç–≤–æ–ª—é—Ü–∏–∏ –ø–µ–Ω—ã

–≥–¥–µ –Ω–æ–≤—ã–π —á–ª–µ–Ω **–≥–µ–π–∑–µ–Ω–±–µ—Ä–≥–æ–≤—Å–∫–æ–≥–æ –±–∞—Ä—å–µ—Ä–∞**:

## 5. –≠–Ω—Ç—Ä–æ–ø–∏–π–Ω—ã–π –±–∞–ª–∞–Ω—Å –∏ –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ –∑–∞–¥–∞—á–∏

### 5.1. –ö–ª—é—á–µ–≤–æ–µ —É—Ä–∞–≤–Ω–µ–Ω–∏–µ —Å–æ—Ö—Ä–∞–Ω–µ–Ω–∏—è –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–æ–Ω–Ω–æ–≥–æ –±–∞–ª–∞–Ω—Å–∞

–í –ì–†–ê –æ–±–Ω—É–ª–µ–Ω–∫–µ —Å –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–π –µ–¥–∏–Ω–∏—Ü–µ–π –ø–µ–Ω—ã —Ä–∞–∑—É–º–∞ —Å–æ—Ö—Ä–∞–Ω—è–µ—Ç—Å—è —Ñ—É–Ω–¥–∞–º–µ–Ω—Ç–∞–ª—å–Ω–æ–µ —Å–æ–æ—Ç–Ω–æ—à–µ–Ω–∏–µ:

–≥–¥–µ:

- $K(G_0)$ ‚Äî **–∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ –∑–∞–¥–∞—á–∏** (–∏–Ω–≤–∞—Ä–∏–∞–Ω—Ç)

### 5.2. –°–≤—è–∑—å —Å –º–∏–Ω–∏–º–∞–ª—å–Ω–æ–π –µ–¥–∏–Ω–∏—Ü–µ–π –ø–µ–Ω—ã

–ü—Ä–∏ –æ–ø—Ç–∏–º–∞–ª—å–Ω–æ–º —Å–æ—Å—Ç–æ—è–Ω–∏–∏ $\Psi^*$:

$$H^c(\Psi^*) = K(G_0) - H_{\min}$$
